# 羽毛球 Magnus 效应 Recipe

## 思路

羽毛球虽然不是球体，但其自然旋转（羽毛裙部旋转）和切削击球会产生短暂的 Magnus 效应，导致轨迹侧向偏移。最新实验表明，在切削球和左右手差异下可观察到显著偏移。本 Recipe 从轨迹偏移中辨识旋转系数 $k_m$，提升羽毛球轨迹预测精度。

## 核心原理

### Magnus 力模型

$$F_m = k_m (\omega \times v)$$

其中 $\omega$ 是旋转角速度，$v$ 是飞行速度，$k_m$ 是 Magnus 系数。

### 羽毛球旋转的特殊性

与乒乓球不同，羽毛球旋转有以下特点：

1. **自然旋转**：羽毛裙部在飞行中因气动效应自然旋转，$\omega$ 与 $v$ 方向近似平行，Magnus 力接近零。
2. **切削旋转**：切削击球产生横向旋转，$\omega$ 与 $v$ 有正交分量，产生侧向 Magnus 力。
3. **旋转衰减快**：羽毛裙部阻力大，旋转在飞行中快速衰减，Magnus 效应主要在击球后短期内显著。
4. **左右手差异**：右手切削和左手切削产生相反方向的旋转，轨迹偏移方向相反。

### 辨识策略

由于 $\omega$ 难以直接观测，采用间接辨识方法：

1. **零旋转基线**：先拟合无旋转模型（只有阻力+重力），建立基线轨迹。
2. **偏移检测**：比较实际轨迹与基线轨迹的侧向偏移。若偏移超过噪声阈值，判定存在旋转效应。
3. **$k_m$ 拟合**：从偏移量反推 $k_m$，假设 $\omega$ 的典型幅值（如 50–200 rad/s）。
4. **分段建模**：Magnus 效应在飞行前段显著、后段衰减，建议分段时间建模。

## 步骤

1. **采集数据**：用高速相机记录切削球和正常球的飞行轨迹，标注击球类型。
2. **基线拟合**：对正常球（无切削）拟合 $k_d$，建立零旋转基线。
3. **偏移分析**：计算切削球轨迹与基线的侧向偏移，统计偏移量与球速的关系。
4. **$k_m$ 辨识**：从偏移量反推 $k_m$，可用 CasADi NLS 批量拟合。
5. **旋转衰减建模**：拟合旋转衰减时间常数 $\tau_\omega$，模型化为 $\omega(t) = \omega_0 e^{-t/\tau_\omega}$。
6. **在线集成**：将 $k_m$ 和 $\tau_\omega$ 加入 ShuttleFlightModel，在 EKF 中作为可辨识参数。

## 关键参数

| 参数 | 建议值 | 说明 |
|---|---|---|
| Magnus 系数 $k_m$ | 0.001–0.01 | 远小于球体的 0.01–0.1 |
| 初始旋转 $\omega_0$ | 50–200 rad/s | 切削击球产生 |
| 旋转衰减 $\tau_\omega$ | 0.05–0.2 s | 羽毛裙部阻力导致快速衰减 |
| 偏移检测阈值 | 3σ | 超过噪声 3 倍标准差 |
| 辨识数据量 | > 50 条轨迹 | 覆盖不同切削角度 |

## 验收指标

| 指标 | 建议 |
|---|---|
| 切削球侧向偏移预测误差 | < 20% |
| $k_m$ 辨识重复性 | 跨批次标准差 < 30% |
| 正常球预测精度退化 | < 5%（加入 Magnus 模型后） |
| 左右手偏移方向正确率 | > 90% |
