# Georgia Tech 异步多相机 Recipe

## 思路

传统多相机融合要求所有相机帧同步（硬件触发或软件对齐），但球类机器人场景中相机帧率、曝光时间和传输延迟往往不同。Georgia Tech 的方案显式针对异步多相机，不需要成对同步，每个相机独立贡献观测因子，通过因子图增量融合。

## 核心原理

### 异步观测模型

相机 $i$ 在时刻 $t_i$ 观测到球位置 $p_b$，观测方程：

$$z_i = \pi(T_{WC_i}^{-1} \cdot p_b(t_i)) + n_i$$

其中 $T_{WC_i}$ 是世界系到相机 $i$ 的变换，$n_i$ 是观测噪声。不同相机的 $t_i$ 不同，不需要同步。

### 连续时间状态表示

球的运动用连续时间模型（如 B-spline 轨迹或物理模型积分）表示，使得任意时刻 $t_i$ 的球状态都可以插值获得：

$$p_b(t) = \text{interp}(t; \text{trajectory})$$

这样，不同相机的异步观测可以直接与连续轨迹比较，无需离散时间对齐。

### 增量因子图融合

每个相机观测到达时：
1. 根据观测时间戳 $t_i$，在连续轨迹上插值得到预测观测。
2. 计算残差 $r_i = z_i - \hat z_i$。
3. 将残差因子插入因子图，触发增量更新。

## 步骤

1. **初始标定**：用 Kalibr 标定各相机内参、相机间外参和时间偏置。
2. **连续轨迹初始化**：用第一批同步观测初始化球的 B-spline 轨迹或物理模型状态。
3. **异步因子添加**：每个相机观测到达时，按自身时间戳创建因子，插入因子图。
4. **增量求解**：ISAM2 增量更新，获取融合后的状态估计和协方差。
5. **时间偏置精化**：将相机间时间偏置作为可优化变量，随运行持续精化。
6. **丢帧处理**：某相机丢帧时，其他相机因子继续提供约束，状态估计不中断。

## 关键参数

| 参数 | 建议值 | 说明 |
|---|---|---|
| B-spline 阶数 | 4（三次） | 平滑且局部支撑 |
| 控制点间隔 | 5–20 ms | 平滑度与精度折中 |
| 因子图滑动窗口 | 100–500 ms | 覆盖最近轨迹段 |
| 时间偏置初值 | Kalibr 标定结果 | 在线精化起点 |
| 观测噪声 | 按距离分段 | 近处小、远处大 |

## 验收指标

| 指标 | 建议 |
|---|---|
| 异步融合位置精度 | < 5 mm（1 m 距离内） |
| 丢帧容忍 | 单相机丢帧 50% 时精度退化 < 15% |
| 时间偏置精化精度 | < 0.5 ms |
| 增量更新耗时 | < 1 ms |
| 多相机一致性 | 重投影误差 < 1 px |
