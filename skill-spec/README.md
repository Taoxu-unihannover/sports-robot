# Skills 工程化实战教程

本目录是《Skills 工程化实战教程：从入门到生产级落地》的书稿工程。

## 本书定位

这是一本 Skills 专项技术书。主线不是泛泛讨论 Agent，而是回答一个更具体的问题：如何把一个可复用任务流程写成可触发、可测试、可分发、可治理、可演化的 Skill。

## 参考材料

- `../写好 Skills 的教程与书稿草案研究报告.md`：主结构来源，提供生命周期、项目研究、模板、测试套件和案例库。
- `../智能体技能设计与实现指南.docx`：理论压缩来源，提供 SKILL.md 解构、渐进式披露、PEV 循环、Harness Engineering 和安全审计案例。
- `../Skill 工程化实战教程-从入门到生产级落地.md`：教程表达来源，提供入门路径、七大原则、案例写法、企业落地和优化章节。
