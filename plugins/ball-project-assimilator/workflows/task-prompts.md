# Task Prompts

## 技术栈拆解

```
扫描项目 {project_path}，识别以下技术栈：
- 仿真平台（MuJoCo/Isaac/PyBullet/Gazebo/ROS2）
- 感知栈（检测/跟踪/状态估计/相机标定）
- 建模栈（球动力学/反弹/旋转/空气阻力/接触模型）
- 控制栈（MPC/IK/RL/IL/QP/轨迹优化/技能库）
- 执行栈（机械臂/移动底盘/发球机/伺服驱动/安全边界）
- 工程栈（训练框架/日志/配置/评估/可视化/CI）

输出技术栈图谱和 skill 缺口报告。
```
