## 5. 数据流

```
Camera 0 ──→ Detector ──→ Tracker ──┐
Camera 1 ──→ Detector ──→ Tracker ──┼──→ Filter ──→ Geometry ──→ 3D State
Camera N ──→ Detector ──→ Tracker ──┘
```

### 5.1 接口定义

| 接口 | 类型 | 说明 |
|------|------|------|
| Image → Detector | np.ndarray (H,W,3) | BGR 图像 |
| Detector → Tracker | Optional[DetectionResult] | 2D 检测结果 |
| Tracker → Filter | Optional[TrackPoint] | 平滑 2D 坐标 |
| Filter → Geometry | BallState | 滤波状态 |
| Geometry → Output | np.ndarray (3,) | 3D 世界坐标 |

## 6. 精度策略

### 6.1 精度目标

| 指标 | 目标值 | 测量方法 |
|------|--------|---------|
| 检测 mAP@0.5 | > 0.8 | 标注数据集 |
| 跟踪 MOTA | > 0.7 | 标注轨迹 |
| 位置 RMSE | < 5 px | 合成数据 |
| 3D 误差 | < 5 cm | 已知位置目标 |

### 6.2 误差预算

| 模块 | 误差来源 | 预算 |
|------|---------|------|
| Detector | 检测偏差 | ±2 px |
| Tracker | 平滑偏差 | ±1 px |
| Filter | 估计偏差 | ±2 px |
| Geometry | 三角化偏差 | ±2 cm |
