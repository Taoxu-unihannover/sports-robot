## 2. MuJoCo 仿真层

### 2.1 场景构建

球类机器人 MuJoCo 场景包含以下元素：

- **地面**：平面 geom，摩擦系数可配置
- **球场**：网球/羽毛球/乒乓球场地边界线
- **机器人**：麦克纳姆底盘/人形/机械臂 URDF/MJCF
- **球体**：网球/羽毛球/乒乓球，带物理属性
- **目标**：可视化球体标记击球目标位置
- **相机**：固定/跟踪相机用于渲染和感知

### 2.2 资产管理

```
assets/mujoco/
  tennis_world/
    scene.xml          # 主场景文件
    robots/
      summit_xls.xml   # 机器人定义
```

### 2.3 中文路径兼容

MuJoCo 不支持中文路径，解决方案：

1. 检测路径是否包含非 ASCII 字符
2. 将资产目录拷贝到 `tempfile.mkdtemp()` 临时目录
3. 使用临时路径加载模型
4. 环境关闭时清理

## 3. Gymnasium 环境层

### 3.1 标准接口

```python
class NavigationEnv(MujocoEnv):
    def __init__(self, xml_file, config):
        ...
    
    def reset_model(self):
        # 初始化机器人位置、目标位置、球速度
        return self._get_obs()
    
    def step(self, action):
        # 执行动作、更新球位置、计算奖励、检查终止
        return obs, reward, terminated, truncated, info
    
    def _get_obs(self):
        # 构造观测向量
        return obs
```

### 3.2 观测空间设计

| 观测 | 维度 | 归一化 | 说明 |
|---|---|---|---|
| 目标距离 | 1 | / diagonal | 到目标的欧氏距离 |
| 目标角度 | 1 | / 2π | 目标方向角 |
| 相对角度 | 1 | / π | 目标相对机器人朝向 |
| 机器人朝向 | 1 | / 2π | yaw 角 |
| 线速度 | 3 | / max_vel | vx, vy, vz |
| 角速度 | 3 | / max_ang_vel | wx, wy, wz |
| 球速度 | 2 | / max_vel | 网球 vx, vy |

### 3.3 动作空间

- 类型：`Box(-1, 1, shape=(3,))`
- 映射：[x_move, y_move, rotate] → 麦克纳姆轮逆运动学 → 四轮力矩

### 3.4 奖励函数

```python
reward = (
    distance_incentive * (prev_distance - curr_distance)  # 接近奖励
    + heading_bonus * cos(angle_to_goal)                   # 朝向奖励
    + velocity_alignment * dot(v_robot, dir_to_goal)       # 速度对齐
    + success_bonus * reached                               # 成功奖励
    + failure_penalty * failed                              # 失败惩罚
    + time_penalty                                          # 时间惩罚
)
```

### 3.5 终止条件

- **成功**：机器人到目标距离 ≤ goal_tolerance (0.2m)
- **失败**：机器人出界 / 网球出界 / 超时

## 4. RL 训练层

### 4.1 算法配置

| 算法 | 推荐场景 | 超参 |
|---|---|---|
| SAC | 连续控制，探索性强 | lr=5e-4, buffer=3M, batch=512, net=[1024,512] |
| PPO | 稳定训练 | lr=3e-4, batch=256, net=[256,256] |
| TD3 | 确定性策略 | lr=5e-4, buffer=1M |
| DDPG | 简单连续控制 | lr=1e-3, buffer=1M |

### 4.2 训练流程

```
1. 注册环境 → gym.make("TennisNavigation-v1")
2. 创建模型 → SAC("MlpPolicy", env, **config)
3. 训练 → model.learn(total_timesteps=N, callback=callbacks)
4. 保存 → model.save("saved_models/model_name")
5. 评估 → model.predict(obs, deterministic=True)
```

### 4.3 并行环境

```python
from stable_baselines3.common.vec_env import make_vec_env

env = make_vec_env("TennisNavigation-v1", n_envs=4, seed=42)
```

### 4.4 Checkpoint 与继续训练

```python
# 保存
model.save("saved_models/checkpoint_5000")

# 加载继续训练
model = SAC.load("saved_models/checkpoint_5000", env=env)
model.learn(total_timesteps=10000)
```

## 5. 评估与可视化层

### 5.1 评估指标

| 指标 | 计算方式 | 说明 |
|---|---|---|
| 成功率 | reached / total_episodes | 到达目标的比例 |
| 平均 reward | mean(episode_rewards) | 平均回合奖励 |
| 平均距离 | mean(final_distances) | 平均最终目标距离 |
| 平均步数 | mean(episode_steps) | 平均回合步数 |
| 训练吞吐 | steps / wall_time | 训练速度 |

## 7. 已知差距与优化方向

| 差距 | 原因 | 优化方向 |
|---|---|---|
| reward 尺度不一致 | distance_incentive 权重不同 | 对齐奖励函数参数 |
| 距离偏高 | 5000 步训练不足 | 增加训练步数到 15000+ |
| 吞吐略低 | 临时目录 I/O 开销 | 使用 from_xml_string 替代 |
